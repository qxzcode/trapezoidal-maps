Where Am I?
An Explanation of Trapezoidal Maps

Theo Lincoln and Quinn Tucker

Overview

Determining which planar region a point belongs to has many applications, such as localizing points to zip codes, detecting collisions in sensory regions for spatial data structures, or UI interaction. However, solving this problem with unprocessed polygonal data would be polynomial in time. Trapezoidal maps provide a data structure that allows the problem to be solved in O(log(n)) time, and our project aims to implement the algorithm as well as create a pedagogical aid to demonstrate it in action. In our tool, users will either choose from a predetermined set of maps, or draw in their own polygonal regions. There will be buttons to move through the steps of the algorithm, as well as a button to play the animation to completion.
